# ProTask
ProTask is a project planing and time tracking application for agile teams.

## Installation
```
git clone https://github.com/Palinana/protask.git
cd protask
npm install
npm run serve
```

## Technologies
Front-end: Vue, Vuetify</br>
Back-end: Node.js, Firebase, Firestore
