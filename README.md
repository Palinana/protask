# ProTask
ProTask is a project planning and time tracking application for agile teams.

## Installation
```
git clone https://github.com/Palinana/protask.git
cd protask
npm install
npm run serve
```

## Technologies
<b>Front-end:</b> Vue, Vuetify</br>
<b>Back-end:</b> Node.js, Firebase, Firestore
